Ce este JavaScript :
* Limbaj de programare orientat pe obiect
* Limbaj folosit pentru introducerea unor functionalitati in paginile web
* Limbaj rulat de catre browser
Script ce se ruleaza in interiorul browser-ului pentru a imbunatati performantele acestuia

Ce este HTML :

* Limbaj de marcare utilizat pentru crearea paginilor web
* Limbaj orientat catre prezentarea informatiilor utilizand un agent utilizator HTML (ex. browser)
Limbaj de programare folosit pentru a scrie aplicatii web
Limbaj de marcare orientat pe obiecte
