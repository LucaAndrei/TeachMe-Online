Ce este Node JS ?
* Node JS este o platforma software utilizata pentru a crea aplicatii web scalabile
Node JS este un editor de text pentru script-urile JavaScript
Node JS este o aplicatie ce permite crearea aplicatiilor web

Ce este Angular JS ?
* Angular JS este un framework JavaScript, o librarie scrisa in JavaScript
Angular JS este un software dedicat paginilor HTML si permite manipularea tag-urilor HTML cu ajutorul limbajului de programare JavaScript
Angular JS este un limbaj de programare creat si intretinut 100% de Google
